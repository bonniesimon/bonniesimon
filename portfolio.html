Corona.info

corona.info | View on Github

corona.info is a web app made using React and Chartjs, designed and developed by me. And all designing is done with css (no frameworks). The application displays covid based info, country wise, on query. Also charted a graph using chartjs based on the data of India.

ICMACE

icmace.in

ICMACE is an international conference hosted by MACE, kothamangalam. The conference required a website for people to know about the event and to register for it. Along with a team, I have developed the website. The modern designed website is the gateway for people to know about ICMACE. The website was designed and coded using modern frontend web technologies.

LockDown quiz app

lockdown-takshak.herokuapp.com

Lockdown is an escape room event held at Takshak, the annual national tech fest of MACE. The event needed a website which acted as a chance to earn 100% cashback. I created a simple riddle game that allow contestants to redeem the 100% cashback. The site was developed using Nodejs and Express along with few modern frontend technologies.

View on Github

Flippy is a price tracker that helps you track prices of products on Flipkart.It is basically a web scraper that logs data onto a json file. The data is obtained from the json file and displayed using chartjs along with few modern frontend technologies.
